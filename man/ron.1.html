ron(1)

NAME

ron -- the opposite of roff

SYNOPSIS

ron [ OPTIONS ]
ron --build FILE ...
ron --install FILE ...

DESCRIPTION

ron is a humane roff(1); for man pages.

OPTIONS

ron expects Ron formatted text on stdin and writes roff formatted text on stdout.

--roff

Generate roff output. This is the default.

--html

Generate HTML output.

-O

Open the HTML version in a browser immediately after
